A preprocessing action should return a new system of equations, and optionally a new set of unknowns together with a postprocessing function.

val preprocess_none : Lang.Term.VarSet.t -> AState.equation Base.list -> preprocessing_action_result

An empty preprocessing action.

val preprocess_detuple : Lang.Term.VarSet.t -> AState.equation Base.list -> preprocessing_action_result

Preprocessing action that projects each unknown returning a tuple into a tuple of unknowns and change the equations accordingly. Postprocessing consist of rebuilding the tuples.

val preprocess_deconstruct_if : Lang.Term.VarSet.t -> AState.equation Base.list -> preprocessing_action_result

Preprocessing action that transforms constraints with conditionals into sets of constraints where conditions have been moved in the precondition of the constraint.

val preprocess_factor_subexpressions : Lang.Term.VarSet.t -> AState.equation Base.list -> preprocessing_action_result

For each equation, search within the system of equations whether there is another equation that constrains a subexpression.

val solve : p:AState.psi_def -> AState.equation Base.list -> Syguslib.Sygus.solver_response * (partial_soln, Counterexamples.unrealizability_ctex Base.list) Base.Either.t

Main entry point: solve a system of equations by synthesizing the unknowns. Returns either a solution as a list of implementations for the unknowns (a triple of unknown name, arguments of a function and body of a function) or a list of unrealizability counterexamples.
