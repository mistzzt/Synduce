Module Lang__SygusInterface

module SygusSolver : sig ... end
val rtype_of_sort : Syguslib.Sygus.sygus_sort -> Lang.RType.t Base.option

Translates a sygus sort to a type. Returns Some type if the sort corresponds to a valid type, otherwise None.

val sort_of_rtype : Lang.RType.t -> Syguslib.Sygus.sygus_sort

Translate a type to a sygus sort. If the type is a type variable, it is replaced by the sort Int. A term of unassigned type appearing in a constraint should mean the type does not matter, and it can be soundly replaced by Int.

val requires_dt_theory : Lang.RType.t -> Base.bool

Returns true if the type argument requires the logic to include datatypes to reason about objects of that type.

val logic_of_operators : Lang.Term.OpSet.t -> Base.string

Returns a string corresponding to the logic that allows to reason about all the operators in the set. Does not include the "DT" prefix required for reasoning about datatypes (see requires_dt_theory).

val dt_extend_base_logic : Base.string -> Base.string

Extend the base logic provided as a string to the same logic with datatypes.

val sygus_of_term : Lang.Term.term -> Syguslib.Sygus.sygus_term

Translates a term to a sygus term.

val term_of_sygus : (Base.string, Lang.Term.variable, Base.String.comparator_witness) Base.Map.t -> Syguslib.Sygus.sygus_term -> Lang.Term.term

Translates a sygus-term to a term in an environment. The environment is a map from variable name to variable.

val declare_sorts_of_vars : Lang.Term.VarSet.t -> Syguslib.Sygus.command Base.list

Given a set of variables, returns a list of commands that correspond to the sort declarations necessary of declare these variables. It is useful especially if the variables as of used-defined variant types.

val declaration_of_var : Lang.Term.variable -> Syguslib.Sygus.command

Returns a sygus command of the form (declare-const v sort) from a variable.

val sorted_vars_of_types : Lang.RType.t Base.list -> Syguslib.Sygus.sorted_var Base.list

Given a list of types, returns a list of sorted sygus variables. Returns as many variables as there are elements in the input lists, and a fresh name is used for each created variable.
