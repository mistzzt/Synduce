val list_or_space : sep:unit Fmt.t -> f:'a Fmt.t -> Stdlib.Format.formatter -> 'a list -> unit

list_or_space prints the list using f for each element, and sep for the separator. If the list is empty, prints a space. If the list is not empty, the printed list is surrounded by two spaces.

option_or_space prints the option using f for the data, and sep for the separator. If the option is None, prints a space. If the option is not None, the printed data is surrounded by two spaces.
